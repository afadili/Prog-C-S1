Compiler un programme utilisant MLV sous Linux ou sous MacOS X

Compiler rapidement votre programme

Pour compiler votre programme, nous allons nous inspirer des exemples proposés par la bibliothèque situés dans le répertoire /usr/share/doc/mlv/examples .

Pour compiler votre programme demo.c, recopiez le Makefile situé dans le répertoire /usr/share/doc/mlv/examples/beginner/Makefile dans le le répertoire où se trouve demo.c. (vous pouvez aussi trouver une version de ce makefile à la page : Outils de compilation automatique).

Modifiez le fichier Makefile copié en remplaçant la ligne 

TARGET=...

par: 

TARGET = demo

Pour compiler votre programme, il ne vous reste plus qu'à taper la commande 

make

Les exemples présents dans le répertoire /usr/share/doc/mlv/examples constituent un bon tutoriel pour apprendre à utiliser la bibliothèque. Nous vous conseillons de tous les compiler et de tous les essayer.

Compiler votre programme avec gcc.

La bibliothèque MLV est compatible avec l'outil pkg-config. Vous pouvez donc récupérer les différentes options de compilation en tapant les commandes suivantes:

  • Pour la compilation:
    pkg-config --cflags MLV
  • Pour les flags de l'édition de liens:
    pkg-config --libs-only-other --libs-only-L MLV
  • Pour les bibliothèques de l'édition de lien:
    pkg-config --libs-only-l MLV

Ainsi, si vous avez écrit un programme demo.c, vous pouvez le compiler en tapant la commande suivante:

gcc -Wall `pkg-config --cflags MLV` `pkg-config --libs-only-other --libs-only-L MLV` demo.c `pkg-config --libs-only-l MLV` -o demo

Écrire une telle commande s'avère fastidieux, nous vous conseillons donc d'utiliser un Makefile.

La page Outils de compilation automatique présente deux exemples de makefile permettant d'automatiser la compilation.
