int MLV_get_date ( int *  seconds,
int *  minutes,
int *  hours,
int *  day,
int *  month,
int *  year,
int *  day_of_the_week 
)

Donne la date du jour (seconde/heure/jour/mois/annee).

L'année est donnée depuis l'an 0.

Une valeur nulle peut être passée en paramètre de la fonction. Dans ce cas, le paramètre est ignoré par la fonction.

Paramètres
secondsLe nombre de secondes écoulées depuis le dernier changement de minute.
minutesLe nombre de minutes écoulées depuis le dernier changement d'heure
hoursLe nombre d'heures écoulées depuis minuit.
dayLe quantième du mois.
monthLe nombre de mois écoulés depuis le début de l'année.
yearLe nombre d'années depuis l'an 0.
day_of_the_weekLe jour de la semaine ( 0: Lundi, 1:Mardi, ..., 6:Dimanche).
Renvoie
Renvoie 1 si le décalage heure d'hivers et heure d'été a été pris en compte, 0 si il ne l'a pas été et -1 si l'information n'est pas disponible.
