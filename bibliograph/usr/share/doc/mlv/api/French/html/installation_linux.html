Installer MLV sous Linux

Repérez dans liste qui suit la distribution que vous utilisez, suivez ensuite les instructions d'installation. Si, votre distribution n'est pas présente dans la liste, choisissez alors l'installation à partir du code source. Si vous rencontrez des problèmes avec les paquets d'installation, utilisez alors l'installation à partir du code source. Cette méthode fonctionnera toujours.

  • Debian et Ubuntu
  • Fedora
  • À partir du code source

Debian et Ubuntu

Commencez par installer les bibliothèques sdl, sdl-gfx, sdl-mixer, sdl-ttf, sdl-image, sdl-image, xml2 et glib2.0 en tapant la commande suivante:

sudo apt-get install libsdl-gfx1.2-dev libsdl1.2-dev libsdl-mixer1.2-dev 
sudo apt-get install libsdl-ttf2.0-dev libglib2.0-dev libxml2-dev 
sudo apt-get install libsdl-image1.2-dev

Téléchargez ensuite les paquets de la bibliothèque :

  • libmlv0_2.0.2-1_amd64.deb
  • libmlv0-dev_2.0.2-1_amd64.deb

(si vous utilisez un ordinateur 32 bits, vous devez utilisez les paquets libmlv0_2.0.2-1_i386.deb et libmlv0-dev_2.0.2-1_i386.deb à la place)

et installez-les en tapant la commande suivante:

sudo dpkg -i libmlv0_2.0.2-1_amd64.deb libmlv0-dev_2.0.2-1_amd64.deb

La bibliothèque MLV est installée.

N'oubliez pas que pour compiler vos programme, il vous faut les outils de compilation. Si vous n'avez pas encore installé ces outils, tapez la commande suivante:

sudo apt-get install build-essential

Fedora

Nous supposerons que vous avez installer une ferdora 64 bits. Les utilisateurs utilisant une fedora 32 bits, devroint adapter le tutoriel en remplacant, la chaîne de caractere "x86_64" par "i686".

Commencez par télécharger les paquets de la bibliothèque suivants:

  • mlv-2.0.2-0.fc22.x86_64.rpm
  • mlv-devel-2.0.2-0.fc22.x86_64.rpm
  • mlv-static-2.0.2-0.fc22.x86_64.rpm
  • RPM-GPG-KEY-Boussicault

et installez-les en tapant la commande suivante:

sudo yum update
sudo rpm --import RPM-GPG-KEY-Boussicault
sudo yum install mlv-2.0.2-0.fc22.x86_64.rpm
sudo yum install mlv-devel-2.0.2-0.fc22.x86_64.rpm
sudo yum install mlv-static-2.0.2-0.fc22.x86_64.rpm

La bibliothèque MLV est installée.

À partir du code source

Avant de commencer à compiler et installer la bibliothèque MLV,il faut que vous vérifiez que vous avez les bibliothèques de développement et les programmes suivants correctement installés:

  • gcc
  • libtool
  • make
  • autoconf
  • doxygen
  • sdl1.2
  • sdl-gfx
  • sdl-mixer1.2
  • sdl-ttf
  • glib2.0
  • xml2

Téléchargez le code source de la bibliothèque MLV : mlv-2.0.2.tar.gz.

Décompressez le code en tapant la commande suivante :

tar -xzf mlv-2.0.2.tar.gz

Installez la bibliothèque en tapant les commandes suivantes:

cd mlv-2.0.2
./configure --prefix=/usr
make
sudo make install

Voilà, la bibliothèque est installée.
