Télécharger la bibliothèque MLV

La dernière version stable de la bibliothèque est la version 2.0.2.

La bibliothèque MLV est diponible sous plusieurs formats et pour différentes plates-formes:

Paquets debian 32 bits :

  • binaires: libmlv0_2.0.2-1_i386.deb
  • développement: libmlv0-dev_2.0.2-1_i386.deb

Paquets debian 64 bits :

  • binaires: libmlv0_2.0.2-1_amd64.deb
  • développement: libmlv0-dev_2.0.2-1_amd64.deb

Paquets RPM pour Fedora 32 bits :

  • binaires: mlv-2.0.2-0.fc22.i686.rpm
  • développement: mlv-devel-2.0.2-0.fc22.i686.rpm
  • bibliothèques statiques: mlv-static-2.0.2-0.fc22.i686.rpm
  • code source: mlv-2.0.2-0.fc22.src.rpm
  • Clé: RPM-GPG-KEY-Boussicault

Paquets RPM pour Fedora 64 bits :

  • binaires: mlv-2.0.2-0.fc22.x86_64.rpm
  • développement: mlv-devel-2.0.2-0.fc22.x86_64.rpm
  • bibliothèques statiques: mlv-static-2.0.2-0.fc22.x86_64.rpm
  • code source: mlv-2.0.2-0.fc22.src.rpm
  • Clé: RPM-GPG-KEY-Boussicault

Windows :

  • Binaires : mlv-2.0.2-win32.zip
  • Projet Code::Blocks : mlv-2.0.2-code_blocks.zip
  • Projet dev-cpp : mlv-2.0.2-dev-cpp.zip

Paquets MacOS 10.10:

  • Paquet d'installation : mlv-2.0.2_macos-10.10.pkg
